a simple & easy static blog framework

>Wow , such a easy static blog framework.  --Isaac·Two·Dog

# Install

```shell
npm install shark-blog -g
```

# Quick Start

create a new blog and start a local server for preview.

```shell
blog create my-blog
cd my-blog
blog start
```
